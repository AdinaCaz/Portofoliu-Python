# Analizator de Compresie Text (Huffman vs Shannon–Fano)

# Descriere
  Acest proiect analizează un fișier text și aplică concepte fundamentale din teoria informației, precum entropia Shannon, codarea Huffman și codarea Shannon–Fano. Proiectul evaluează eficiența compresiei și vizualizează atât arborii de codare, cât și distribuția frecvențelor caracterelor.

# Funcționalități
- Analiza frecvenței literelor
- Calculul distribuției probabilităților
- Calculul entropiei Shannon
- Implementarea algoritmului Huffman
- Implementarea algoritmului Shannon–Fano
- Vizualizarea arborilor pentru ambele metode
- Grafic tip bară pentru frecvența literelor
- Compararea eficienței și redundanței
